Lecture 28 - Numerical Solutions of First-Order Systems¶

Overview, Objectives, and Key Terms¶

This lesson is a direct extension of Lecture 27, in which the focus was the numerical solution of first-order IVPs. Here, those techniques are extended to systems of first-order equations.

Homework 11 Recap¶

def C_p(T):
    """Heat capacity for O_2 as given in problem statement.
    """
    A = 31.32234; B=-20.23531; C=57.86644; D=-36.50624; E =-0.007374
    t = T/1000
    return A + B*t + C*t**2 + D*t**3 + E/t**2

def mean_C_p():
    """Compute mean of C_p(T) over T in [100, 700]
    """
    return quad(C_p, 100.0, 700.0)[0]/600.0

def T_C_p_avg():
    """Determine temperature T such that C_p(T) equals mean(C_p(T)).
    """
    return fsolve(lambda x: mean_C_p() - C_p(x), x0=400.0)[0]

def model_mu(T, coeffs):
    """Model for mu(T) with coefficients.
    """
    a, b, c = coeffs
    return b*np.exp(a*T) + c

def get_best_coeffs():
    """Generate the best coefficients for model_mu using curve_fit.
    """
    T, nu, mu = read_viscosity()
    a, b, c = -0.01, 0.016, 0
    foo = lambda w, x, y, z: model_mu(w, [x, y, z])
    a, b, c = curve_fit(foo, T, mu, p0=(a, b, c))[0]
    return a, b, c

a, b, c = get_best_coeffs()
best_coeffs = [a, b, c]

Objectives¶

By the end of this lesson, you should be able to

  • Set up and solve simple linear systems numerically using NumPy.
  • Solve systems of first-order IVPs numerically using forward and backward Euler’s method
  • Solve systems of first-order IVPs using odeint

Prerequisites¶

You should already be able to

  • Solve IVPs using Euler’s method based on the material of Lecture 27
  • Define one- and two-dimensional arrays using NumPy arrays based on the material of Lecture 4.
  • Use np.linalg.solve to solve \(\mathbf{Ax = b}\) based on the material of Lecture 4.

Please review these topics (and resources) as needed.

Key Terms¶

  • odeint
  • first-order system

Example: Solve the following system

\[\begin{split}\left [ \begin{array}{rrrrr} -2 & 1 & 0 & \ddots \\ 1 & -2 & 1 & 0 & \ddots \\ 0 & 1 & -2 & 1 & \ddots \\ & & & \ddots & \\ & & & 1 & -2 \end{array} \right ] \left [ \begin{array}{c} x_0 \\ x_1 \\ \vdots \\ x_n \\ \end{array} \right ] = \left [ \begin{array}{c} 1 \\ 1 \\ \vdots \\ 1 \end{array} \right ]\end{split}\]

Example: For \(\Delta = 0.1\), compute \(x(10)\) and \(y(10)\) by solving the following system using (1) forward Euler, (2) backward Euler, and (3) odeint:

\[\begin{split}\left [ \begin{array}{c} x' \\ y' \end{array} \right ] = \left [ \begin{array}{rr} -10 & 0.00 \\ 10 & -0.01 \\ \end{array} \right ] \left [ \begin{array}{c} x(t) \\ y(t) \end{array} \right ] + \left [ \begin{array}{c} 1 \\ 0 \end{array} \right ]\end{split}\]

for \(x(0) = y(0) = 0\).
